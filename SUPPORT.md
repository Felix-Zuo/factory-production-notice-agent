# Support

This repository is a public, local-first demo. It does not provide production
support for private operational deployments.

## Good Public Questions

- A reproducible bug using synthetic JSON from `sample_data/`.
- A reusable adapter or workflow idea that fits the generic operations notice
  contract.
- Documentation or packaging issues in a fresh checkout.

## Not Public Support

- Private customer data review.
- Real production, warehouse, maintenance, or service records.
- Deployment of the local demo API as an authenticated production service.

For suspected security issues, follow [SECURITY.md](SECURITY.md).
