成熟度信号

Maturity Signals

仓库把安全默认值、验证门禁、CI、发布记录和贡献材料作为产品的一部分维护。

The repository treats safety defaults, validation gates, CI, release notes, and contribution materials as part of the product.

1Preflight验证必填字段、数值、布尔值和导入契约。Validates required fields, numbers, booleans, and import contracts.
2Spreadsheet Safety写入 Excel 前转义公式型文本和控制字符。Escapes formula-like text and control characters before writing Excel.
3Local Boundary本地 API 默认只绑定 loopback，避免误暴露。The local API binds to loopback by default to avoid accidental exposure.
4CI Gate自动覆盖测试、样例验证、导入、生成和包内容检查。CI covers tests, sample validation, import, generation, and package checks.
5Human Approval工具准备发布包，真实作业放行仍由人工确认。The tool prepares release packets while real operational release stays human-approved.
