Factory Production Notice Agent

生产通知单自动化工作台

Production Notice Automation Workbench

把工单、产品、物料、工艺、包装和质检要求收敛成一份标准请求，自动生成 Excel 通知单、浏览器预览、发布清单和可编排上下文。

A manufacturing workflow that converts work-order, product, material, routing, packaging, and quality data into release-ready notice artifacts.

核心能力

Core Capabilities

结构化工单

Structured Work Order

用标准 JSON 承接订单、产品、数量、交期、物料、路线、包装和质检字段。

A stable JSON contract for order, product, quantity, due date, material, routing, packaging, and quality fields.

通知单生成

Notice Generation

同一份请求同时输出 Excel 工作簿、网页预览、发布清单和结构化上下文。

One payload produces the Excel workbook, browser preview, release manifest, and structured context.

现场审核

Shop-floor Review

物料需求、工艺路线、包装规则、质检要求和审批区集中在一张可审核页面。

Material demand, routing, packaging, quality checks, and approvals stay in one review surface.

Agent 接口

Agent Interface

命令行、本地 HTTP API、工作流文件和 Skill 文件都围绕同一份契约展开。

CLI, local HTTP API, workflow docs, and skill instructions share the same contract.

输出资产

Generated Artifacts

生成结果包含通知单工作簿、浏览器预览、manifest 和 agent context。生产、质检、仓库、计划可以围绕同一份数据进行审核和流转。

The output set includes a notice workbook, browser preview, manifest, and agent context. Production, quality, warehouse, and planning teams work from the same payload.

Automation Contract

Expansion Path
