Factory Production Notice Agent

A compact production-notice generator for manufacturing workflows. It turns a structured order request into an Excel notice, a browser preview, and an agent-readable context payload.

一个面向制造现场的生产通知单生成工具：用标准 JSON 输入替代大量固定模板，输出 Excel、网页预览和智能体可读取的结构化上下文。

Why It Exists

Many factories still release work orders through spreadsheet templates. This project keeps the template logic in code, keeps the public demo data synthetic, and gives external agents a stable interface.

许多工厂仍然通过表格模板下发生产任务。本项目把模板逻辑收敛到代码里，用合成数据做展示，并给外部 Agent 留出稳定接口。

Capabilities

Notice Generation

Generate styled Excel notices and HTML previews from normalized production requests.

Agent Contract

Expose CLI, HTTP endpoints, sample data, schema paths, and workflow rules in agent_interface.json.

Local First

Run on a single workstation, keep sensitive BOM and notice history outside the repository.

Integration Path

  • ERP, MES, WMS, BOM, or scheduling adapters can produce the standard notice request JSON.
  • The generator owns the layout, preview, manifest, and structured agent context.
  • A human reviewer can approve final release before the notice is sent to production.
